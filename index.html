Conceptos de JavaScript

Callback

Un callback es una función que se pasa a otra función como argumento y se ejecuta después de que se completa la primera función.

Promise

Una Promise es un objeto que representa la eventual finalización (o falla) de una operación asíncrona y su valor resultante.

Async

Async es una palabra clave que se usa para declarar una función asíncrona, la cual devuelve una Promise.

Await

Await es una palabra clave que se usa dentro de funciones asíncronas para esperar la resolución de una Promise.
